# **App Name**: AI LekiOOS Studio

## Core Features:

- AI Code & Component Builder: Utilize an integrated AI tool to generate code snippets and basic UI components directly from natural language text descriptions.
- Integrated Code Editor: Provide a Monaco Editor instance for live code editing with essential features like syntax highlighting and basic auto-completion.
- Interactive Live Preview: Display a real-time, interactive preview of the application being built, including responsive mobile and desktop viewports.
- File Manager & ZIP Export: Allow drag & drop file uploads and a simplified visual file system, with the ability to export the current project as a ZIP archive.
- User Authentication & Cloud Sync: Enable user registration and login, with cloud synchronization for saving and accessing user projects.
- Multilingual UI Toggle: Offer a seamless language switcher to display the application interface in Russian (ru), English (en), and Lezghian (lez).
- Dynamic Theme Customization: Empower users to personalize the app's visual theme by selecting custom UI colors via an integrated color picker.

## Style Guidelines:

- The primary color is a strong, tech-infused blue (#2E52B8), symbolizing intelligence and precision for key actions and branding against a dark theme.
- The background features a very dark, desaturated blue (#101317) to create a focused and professional environment conducive to long periods of work.
- A vibrant sky blue accent color (#3CC8E6) will be used for interactive elements, highlights, and status indicators, providing clear visual contrast and a modern feel.
- Headlines and prominent text will use 'Space Grotesk' (sans-serif) to convey a modern, technical, and scientific aesthetic fitting for an AI builder.
- Body text and general UI copy will be set in 'Inter' (sans-serif), ensuring excellent readability and a clean, objective appearance across the application.
- Code display within the editor and log outputs will utilize 'Source Code Pro' (monospace) for optimal code legibility.
- Icons should be modern, outline-based, and highly intuitive, with a consistent style that complements the tech-focused and minimalistic UI. Icons should clearly communicate file actions, project settings, and build statuses.
- The layout will emphasize efficiency and organization, featuring responsive, adaptable split-pane designs for editor, preview, and file management areas, optimized for various screen sizes including specified mobile viewports.
- Subtle and purposeful animations will be employed for state changes, modal transitions, and interactive feedback, enhancing the user experience without being distracting. Loading indicators and progress bars will provide clear visual cues during background processes.
