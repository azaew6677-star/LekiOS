\# AI LekiOOS Studio



Добро пожаловать в AI LekiOOS Studio — мощную среду разработки на базе искусственного интеллекта, созданную для ускорения создания современных веб-приложений. Этот проект представляет собой стартовый набор на базе Next.js, интегрированный с Firebase, ShadCN для UI-компонентов и Genkit для функций генеративного ИИ.



\## 🚀 Начало работы



Этот проект готов к работе "из коробки". Просто запустите его, и вы попадете в интегрированную среду разработки прямо в вашем браузере.



\- \*\*`src/app/page.tsx`\*\*: Это главная страница, где собраны все ключевые компоненты интерфейса, такие как AI-агент, файловый менеджер и редактор кода.

\- \*\*`src/components/dashboard`\*\*: Здесь находятся основные компоненты пользовательского интерфейса.

\- \*\*`src/ai`\*\*: Этот каталог содержит всю логику для работы с генеративным ИИ с использованием Genkit.



\## ✨ Основные возможности



\- \*\*Генерация кода с помощью ИИ\*\*: Используйте панель AI-агента для генерации кода из текстовых описаний.

\- \*\*Генерация изображений\*\*: Создавайте изображения и логотипы на основе ваших идей.

\- \*\*Интегрированный файловый менеджер\*\*: Просматривайте структуру вашего проекта прямо в приложении.

\- \*\*Редактор кода\*\*: Просматривайте сгенерированный ИИ код.

\- \*\*Живой предпросмотр\*\*: В настоящее время эта функция находится в разработке, но в будущем она позволит вам видеть изменения в реальном времени.

\- \*\*Экспорт на платформы\*\*: Встроенная (имитируемая) интеграция с GitHub Actions для демонстрации того, как могут быть организованы сборки для различных платформ (Android, iOS, Desktop, Web).

\- \*\*Синхронизация с GitHub\*\*: Кнопка для имитации синхронизации вашего проекта с репозиторием GitHub.



\## 🛠 Стек технологий



\- \*\*Фреймворк\*\*: Next.js (App Router)

\- \*\*UI-компоненты\*\*: ShadCN, Tailwind CSS

\- \*\*Генеративный ИИ\*\*: Google Genkit

\- \*\*Бэкенд и база данных\*\*: Firebase (Firestore, Auth)

\- \*\*Язык\*\*: TypeScript



\## 🔧 Кастомизация и развертывание



Вы можете изменять и расширять этот проект по своему усмотрению. Для развертывания вашего приложения вы можете использовать Firebase App Hosting или любую другую платформу, поддерживающую Next.js.





Добавил выпадающее меню: При нажатии на эту кнопку появляется меню, где вы можете выбрать целевую платформу (APK, IPA, EXE и т.д.).

Перенаправление на GitHub Actions: Выбор любого пункта в этом меню откроет новую вкладку браузера и перенаправит вас на страницу GitHub Actions вашего репозитория.

Теперь, когда вы нажмете на любой из вариантов сборки, вас перенаправит на страницу, где могли бы находиться ваши реальные рабочие процессы (workflows) для сборки приложения. Я использовал в качестве примера URL репозитория, который соответствует названию вашего проекта.
